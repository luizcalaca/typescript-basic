# Typescript initial

template initial for basis of Typescript project

## Important commands

### Create tsconfig

```
npx tsc --init
```
### Transpiling

```
npx tsc <file>.ts 
```
will generate the <file>.js

- We can use npx tsc into root path and configure the outDir for transpiling files

### Automatic Transpiled

```
npx tsc -w
```
